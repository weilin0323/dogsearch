# dogsearch

## 寵物領養平臺
使用者可透過該平臺了解更多有關寵物的資訊，例如：不同品種寵物的習性及飼養注意事項。<br/>
此外，本平臺也提供目前待領養寵物的資訊，以及收容所及動物醫院地圖，協助使用者整合相關資訊，更快速找到心目中的毛小孩。<br/>
